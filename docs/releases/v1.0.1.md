## 更新内容

### 课程锁定与抢课顺序

- 可在课程库或课表课程块的右键菜单中选择“锁定”或“解锁”，两种操作使用不同图标。
- 被锁定的课程以独立的灰色样式显示，不再出现在抢课顺序中；锁定状态会随规划数据保存及导入导出。
- 课表导出会同步显示锁定课程的颜色。

### 非本周课程

- 课表现在会显示非本周课程，并在标题前添加“[非本周]”标识。
- 非本周课程使用与普通课程、锁定课程明显区分的浅色样式，并适配浅色、深色和高对比度主题。
- 当课程同时处于锁定和非本周状态时，优先使用非本周样式。
- 课表导出不会包含非本周课程。

### 导入预览

- 导入前的二次确认改为类似 Git 合并结果的汇总，集中展示解析、添加、更新、跳过、冲突、警告和错误。
- 移除高级筛选和额外确认步骤；缺失课程同步等导入选项直接纳入同一确认流程。
- 大型导入结果采用有界预览，优先呈现冲突、错误等重要信息。

### 窗口与界面

- 抢课顺序窗口改为独立窗口，可独立最小化和恢复；主窗口最小化或关闭后仍可继续使用。
- 修复抢课顺序窗口在置顶状态下图标只显示填充部分的问题。
- 规划标签栏的学分显示保留到两位小数。
- 补全非本周课程及导入预览相关的中英文文案。

### 发布与稳定性

- 支持在同一 v1.0.1 发布下通过递增 MSIX 第四段版本号进行原位升级。
- 修复发布工作流计算同版本包修订号的问题，并补充相关自动化测试与文档。

## 安装

下载 `CoursePlanner-v1.0.1-x64.zip`，解压后双击 `Install-CoursePlanner.cmd`。安装程序会验证包签名，首次安装时通过 UAC 将发布者证书加入本地计算机信任库，再为当前用户安装应用。

详细的系统要求、校验和验证及手动安装方式请参阅仓库 README。

## What's changed

### Course locking and registration order

- Courses can be locked or unlocked from the context menu in either the course library or timetable, with distinct icons for the two actions.
- Locked courses use a dedicated gray style and are excluded from registration order. Lock state is persisted with plan data and supported by import and export.
- Timetable image export now reflects the locked-course color.

### Non-current-week courses

- The timetable now shows non-current-week courses with a `[Non-current week]` title prefix.
- These courses use a lighter style clearly separated from normal and locked courses, with light, dark, and high-contrast theme support.
- When a course is both locked and outside the current week, the non-current-week style takes precedence.
- Timetable image export excludes non-current-week courses.

### Import preview

- The second confirmation now presents a Git-merge-like summary of parsed, added, updated, skipped, conflicting, warning, and error results.
- Advanced filters and the extra confirmation step were removed; import options such as synchronizing missing courses are handled in the same confirmation flow.
- Large imports use a bounded preview that prioritizes conflicts, errors, and other important details.

### Windows and interface

- Registration order now opens in an independent window that can be minimized and restored separately and remains usable when the main window is minimized or closed.
- Fixed the pinned-state icon showing only its filled portion.
- Plan-tab credit values now retain up to two decimal places.
- Completed Chinese and English localization for non-current-week courses and import preview.

### Release and reliability

- Added in-place upgrades within the same v1.0.1 release by incrementing the fourth MSIX version component.
- Fixed same-version package revision calculation in the release workflow and expanded automated tests and documentation.

## Install

Download `CoursePlanner-v1.0.1-x64.zip`, extract it, and double-click `Install-CoursePlanner.cmd`. The installer verifies the package signature, uses UAC on first install to trust the publisher for the local computer, and then installs the app for the current user.

See the repository README files for system requirements, checksum verification, and manual installation.
